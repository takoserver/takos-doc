# コミュニティーを検索する

### Endpoint: `GET /takos/v2/client/room/search`

#### Parameters

| Name  | Type   | Description | Required | Default |
| ----- | ------ | ----------- | -------- | ------- |
| query | string | 検索クエリ  | true     |         |

※[]はurl埋め込み

#### Response

```
{
    status: true
}
```

#### Description

コミュニティーを検索します。
